# N8N Calendar Integration

這是一個 n8n 工作流項目，用於整合 Google Calendar 和 Telegram，自動發送每週行程。

## 功能

- 📅 Google Calendar 集成
- 💬 Telegram 通知
- 🤖 AI Hub 支持（記憶和智能回應）
- 🗄️ PostgreSQL 數據庫

## 本地開發

### 前置要求
- Docker 和 Docker Compose
- Git
